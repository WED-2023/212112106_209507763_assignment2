    üöß Challenges We Faced

    Developing this game came with several hurdles, each one a learning opportunity for us. Here are some of the challenges we tackled:

    • Splitting the Game into Modules: One of the toughest parts of this project was organizing the code. We had to carefully split the game logic into manageable modules, like player movement, enemy behavior, and game state management.
    • Choosing the Right Functions: Determining which functions to use and where was critical. We focused on keeping our functions small and reusable, like separating movement from rendering. The logic for bullet collision also had to be modular.
    • Color Combinations: We spent a lot of time testing different color combinations to make sure the game looked appealing but also provided good contrast. Balancing the ship and enemy colors was essential as well for gameplay.
